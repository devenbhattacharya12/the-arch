📁 How to Use These Assets

1. Download all assets by clicking the buttons above

2. Create an assets folder in your Expo project root:

your-expo-app/ ├── assets/ │ ├── icon.png │ ├── splash.png │ ├── adaptive-icon.png │ ├── notification-icon.png │ └── favicon.png ├── app.json └── ...other files

3. Update your app.json to reference these assets

4. Run your build: eas build --platform android --profile preview
